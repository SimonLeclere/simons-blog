## Blocs de code

Code inline : la fonction `getPostBySlug()` accepte un `slug` en paramètre.

Bloc simple :

```ts
const posts = getAllPosts()
console.log(`${posts.length} articles trouvés`)
```

Bloc avec titre et ligne surlignée :

```tsx title="src/components/mdx/figure.tsx" {3}
export default function Figure({ src, caption }: Props) {
  return (
    <figure className="my-8">
      <Image src={src} alt={caption || ''} width={1200} height={630} />
      {caption && <figcaption>{caption}</figcaption>}
    </figure>
  )
}
```
